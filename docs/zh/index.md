# Verse 语言文档

!!! warning
    这是《Verse 之书》的早期草稿。欢迎提出改进建议。频繁的更新是可以预期的。

本文档深入介绍了 Verse 编程语言、其哲学和核心概念。

Verse 是 Epic Games 开发的一种多范式编程语言，借鉴了函数式、逻辑式和命令式编程传统，创建一个连贯的系统来构建元宇宙体验。

Verse 有三个核心原则：

- **不过是代码** - 复杂的概念被表达为原始的 Verse 结构
- **只有一种语言** - 编译时和运行时的结构相同
- **元宇宙优先** - 专为全球模拟环境而设计

!!! note
      本文档对应 Verse 主开发分支的最新版本，一些功能可能在正式发布之前被讨论，因此可能会发生改变。一些 Epic 的内部功能也可能被讨论。
