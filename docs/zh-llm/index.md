# Verse 语言文档

!!! warning
    本文档是《Book of Verse》的早期草稿。欢迎提出改进建议。
    预计会频繁更新。

本文档深入介绍 Verse 编程语言及其设计理念与核心概念。

Verse 是由 Epic Games 开发的多范式编程语言，
融合了函数式、逻辑式和命令式传统，
打造用于构建元宇宙体验的连贯系统。

Verse 遵循三个核心原则：

- **仅是代码**——复杂概念以原始 Verse 构造表达
- **仅一种语言**——编译时和运行时使用相同构造
- **元宇宙优先**——专为全局仿真环境设计

!!! note
      本文档对应 Verse 主开发分支的最新版本，部分功能可能
      在正式发布前被讨论，因此可能会有变化。也可能涉及
      部分 Epic 内部功能。
